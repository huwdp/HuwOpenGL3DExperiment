## Dependencies

 - SDL 2
 - FreeGlut 3

Run commands in Terminal below to install dependencies.

```
sudo apt-get install libsdl2-dev libsdl2-image-dev libsdl2-mixer-dev libsdl2-ttf-dev mesa-utils freeglut3-dev freeglut3 freeglut3-dev
```
